# Contributing to Zyneriq

Thank you for considering a contribution to Zyneriq.

## Getting Started

1. Set up the project using the instructions in `README.md`.
2. Create a branch for your change.
3. Make the change and keep it focused.
4. Run the relevant checks before submitting.

## Style

- Follow the existing Django project structure.
- Keep code readable and narrowly scoped.
- Avoid unrelated formatting churn.
- Add tests for risky or user-facing behavior.
